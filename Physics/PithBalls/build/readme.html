PithBalls

By: Fisher Evans (website)

Source: GitHub

This is a simple program that simulates charged "Pith Balls" in a 2-dimensional vacuum. The only forces acting on the balls are due to electrical charges (F = k*q1*q2/(d^2) or elastic collisions between each other.

The Display

There are three main portions of the screen: the simulation area, the ball information and the world statisitics. You can turn off the ball information and world statistics by pressing F2.

Simulation Area: This is where the actual simulation occurs. This is the main portion of the screen and is "behind" to the other two displays. Each ball is colored, has an ID to the top left of it, and a charge sign to the top right. The more dense the ball, the darker the color. The more charged the ball is, the darker the charge sign is. The ID is only shown when the other two displays are. The balls also have a black line coming out of them. This shows the directiona nd magnitude of their current acceleration due to Coulombs's Law. Behind the balls there is a 1 meter grid.

Ball Information: This is the left portion of the screen. It displays information for each ball (identified by its color and ID) in a box. You can scroll through this list using the page up and page down buttons.

World Statistics: This is the right portion of the screen. It displays information about the current simulation and the configured parameters for the next simulation.

Button Input

Page Up/Down: This will scroll through the ball information.

Arrow Keys: This will scroll through the ball information.

F1: This will pause the sim.

F2: This will Hide the info panes/ball ids.

F3/F4: This will speed up or slow down time.

F5/F6: This will zoom in or out.

F7/F8: This will scale the acceleration line.

F9: This will follow the average postion of the balls.

F10: This will reset the view port.

F12: This will start a new simulation based on the input paramerters.

Command Input

You can enter commands simply by typing, and pressing Enter to execute them. Your input will show up in the grey bar on the bottom of the page. When you execute a command, you will also so red output telling you wheter it was successful or not. There are 2 main settings: ranges and single variables. For ranges you can either you can enter a range like "command 0.1 2.4", or treat it like a ssingle variable "command 8.99E-9".

mass (min max|both) Takes floating points. This will set the range of masses when creating new balls in kg.

vel (min max|both) Takes floating points. This will set the range of intial velocities when creating new balls in m/s/s.

charge (min max|both) Takes floating points. This will set the range of charges when creating new balls in c. Please use positive values only. See the count command for negativity.

radius (min max|both) Takes floating points. This will set the range of radii when creating new balls in m.

rest (min max|both) Takes floating points. This will set the range of restitution to use when creating new balls.

count (neg pos|both) Takes whole numbers. This define how many balls to create when creating a simulation. The neg value is number of negativly charged balls, while the pos is positive. Passing one number will set both the negative and positive counts to passed number.

timescale x Takes a floating point. This will set the time scale.

viewscale x Takes a whole number. This will set the view scale in pixels per meter.

area x Takes a floating point. This will set the spawn area around 0, 0 in m.
